# 2 使用 GitHub 部署个人网站

首先，github 的网站分为两种类型，一种是账户的站点，一种是项目的站点。

项目的站点聚焦于某一个项目，所以他的网站问卷是在项目下，账户的网站是聚焦于这个账户，所以有特定的要求。

下面我将介绍如何部署一个账户网站。

## 2.1 创建项目

在 github 中创建一个新的项目，名称为 `username.github.io`。此处的 `username` 是你的 github 用户名。

一般的，我们建议打开 README 文件。

## 2.3 部署项目文件

随后我们到我们项目的 Code 页面下，复制我们项目的地址。

<img src="/assets/img/2025-11-15/Pasted image 20251115222221.png" height="60%" width="60%">

接下来，我们到我们电脑下的一个特定的目录中，使用 git 配置这个项目。

```powershell
git clone <Press Ctrl+v>
```

我们使用 git 把我们刚刚的网址 clone 下来。随后 cd 到这个目录中去。

这个目录就是我们的网站的目录了，请注意，这部分的操作是 git 的基础操作，所以这里直接使用了指令，而不具体解释命令的含义。请在另外的文件中去学习如何使用 git。

我们在这个目录下创建一个 `index.html` 文件，这个文件就是我们的项目的首页。我们如下创建一个页面作为测试界面。

```html
<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Text</title>
</head>
<body>
    <h1>HelloWorld</h1>
    <p>This is a page of testing.</p>
</body>
</html>
```

随后我们依次使用如下的命令。

```powershell
git add .
git commit -m "Add index.html"
git push origin main
```

这部分就是把我们的项目提交到 github 中去，此时我们去 github 的项目下去看，应该是可以看到我们的项目文件更新了。

## 2.4 访问网站
