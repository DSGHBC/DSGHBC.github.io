在C语言中，默认的基本类型中没有提供变长数组，这意味着我们所使用的数组都是定长的，甚至在c89中，我们不可以通过一个变量来指定一个数组的长度。

不过通过C语言提供的函数，我们可以实现一个构造数据类型，满足我们的要求。

## 可变长的实现原理

基于C的内存构型，我们知道数组的物理存储是连续的一个区域，所以在数组的尾部后一定还有数据，这意味着我们无法使用后续的空间记录元素，这就是C语言不支持动态数组的原理。

不过我们可以通过一个巧妙的方式来实现，我们可以在每次改变数组元素个数的时候，重新定义一个静态的数组即可，通过多次申领一个静态数组，我们就实现了一个逻辑层面的动态数组。

所以这个过程就涉及到了动态的内存分配(扩大或者缩小)，数据的复制等问题。我们将会在后文实现。

# 数据类型设计

## 结构体设计

我们来讨论一个变长数组所具有的属性，除了变量所具有的通用五个属性之外，它应该还具有一个属性，用于表示它所包含的个数。

我们不妨称这个属性为长度，这个属性可以保证我们的数据类型的正常运行。

```c
typedef struct VectorInt{
    int *num;
    int len;
}VectorInt;
```

我们看到了这个数据类型被我们设计出来了，它具有一个`int`类型的指针，一个`int`型的长度。`num`指针指向数组的开头，同时说明了这个数组的类型，是一个整型数组。那么通过这个指针，我们可以借助偏移量去访问整个数组。

以及这里其实存在优化的空间，这里我们后面会讨论这一点。

## 初始化设计

介于C中没有所谓构造函数析构函数的概念，我们使用普通的函数来手动实现这一点，不过对于这样一个简单的函数，我们初始化没有必要使用一个函数了，我们可以借助一个宏来实现我们的目的。

```c
#define VectorIntInit {NULL, 0}
```

这个宏其实就是如下语句的缩写。

```c
VectorInt veci = {NULL, 0};
```

我们使用宏变成了如下的模式。

```c
VectorInt veci = VectorIntInit;
```

使用宏便于我们统一修改构造函数。

## 析构函数设计

由于我们使用一个变长的数组，这意味着，我们需要手动的设定内存的分配，而在C语言中，我们手动分配的内存同样需要我们手动释放，以免出现内存泄露的问题。

所以我们需要设定一个析构函数，当我们使用这个类型结束的时候，我们手动释放他。

```c
void VectorInt_Free(VectorInt *veci)
{
    free(veci->num);
    veci->len = 0;
    veci->num = NULL;
}
```

我们注意到，这个函数，接受了我们的自定义类型作为输入。

他首先释放了指针所指向的内存，随后我们手动重置了指针的指向，设置为空指针，而不是一个野指针。并设定数组长度为0。

# 数据类型操作定义

对于一个数据类型，我们通常会有增删改查这四种基本操作，其他的复杂的操作都基于这四个操作完成，例如排序等。

## 增

对于一个数组，我们的增加操作一般有两种，一个是尾部的增加数据，一个是在特定的位置插入数据。

### 增加元素(append)

```c
void VectorInt_Append(VectorInt *veci, int num)
{
    int *new = (int *)realloc(veci->num, (veci->len + 1) * sizeof(int));

    if (new == NULL)
    {
        printf("error");
        return;
    }
    *(new + veci->len) = num;

    veci->num = new;
    veci->len += 1;
}
```

增加元素的关键在于内存的分配，上文中我们提到了改变内存，来实现动态数组，这里即是。

我们使用了`realloc`函数在`veci->num`处重新分配了一个内存区域，这个内存区域比原数组大1。随后我们做了一个简单的判断，看看指针有没有被分配到内存。

在分配成功之后，我们把这个值提供给最后一位，并把我们更改的数据同步到数组中。

>　请注意，我们务必使用指针传递，否则无法修改动态数组的类型。

### 插入元素(insert)

```c
void VectorInt_Insert(VectorInt *veci, int index, int num)
{
    int *new = (int *)realloc(veci->num, (veci->len + 1) * sizeof(int));

    if (new == NULL)
    {
        return;
    }
    for (size_t i = veci->len; i >= index; i--)
    {
        *(new + i + 1) = *(veci->num + i);
    }
    *(new + index) = num;

    veci->num = new;
    veci->len += 1;
}
```
对于插入算法，我们需要注意的是，我们需要获取插入的位置和插入的值，随后对于前面相同不变的值，就像在append中我们做的那样，我们自然可以简单的赋值。但是对于后面具有偏移的部分，我们需要使用一个for循环来替代简单的赋值。而对于这两个区域，区分的标志就是`index`。

其余的操作与上面的append无异。
