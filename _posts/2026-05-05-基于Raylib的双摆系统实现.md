# 2 渲染单摆

简单的实现思路是，写一个函数去渲染一个摆，随后在调用函数调用它，渲染多个摆。先写出函数签名。

```
void DrawPendulum(Vector2 StartPos,int length,int rangle);
```

对于一个单摆，我们需要三个参数，一个是开始的坐标，一个是摆长，一个是摆角，这样我们就可以刻画一个单摆。

现在来实现一个单摆的渲染，其实就是两个部分，一个是杆，一个是末端的球，所以我们可以找到 Raylib 中的 Draw 部分，查找相关的函数。

![Pasted image 20260505155138](/assets/img/2026-05-05/Pasted image 20260505155138.png)

我们看到两个需要使用的函数，一个是 `Line` 族，一个是 `Circle` 族。

我们使用 LineV 这个函数，创建两个向量和一个颜色，分别查一下对应的定义，然后就可以写了。

![Pasted image 20260505160701](/assets/img/2026-05-05/Pasted image 20260505160701.png)

实现效果如上，需要注意的无非是 `sinf` 和 `cosf` 两个函数需要使用 `float` 版本的。

另外关于角度 `angle`，他是摆杆和水平的偏角，所以我们的公式如下。

$(x',y')=(x+length*sinf(angle),y+length*cosf(angle))$。

关于 `Vector2` 和颜色的定义，在头文件 `raylib.h` 里面都可以查询到，这里不在展开说。

# 3 计算摆角

在第一部分我们给出了一组公式，我们只需要把它变成代码即可。

```cpp
angle1_dd = (-1 * G * (2 * m1 + m2) * sinf(angle1) -
			 m2 * G * sin(angle1 - 2 * angle2) -
			 2 * sinf(angle1 - angle2) * m2 * (angle2_d * angle2_d * L2 + angle1_d * angle1_d * L1 * cosf(angle1 - angle2))) /
			(L1 * (2 * m1 + m2 - m2 * cos(2 * angle1 - angle2)));
angle2_dd = (2 * sinf(angle1 - angle2) * (angle1_d * angle1_d * L1 * (m1 + m2) + G * (m1 + m2) * cosf(angle1) + angle2_d * angle2_d * L2 * m2 * cosf(angle1 - angle2))) /
			(L2 * (2 * m1 + m2 - m2 * cosf(2 * angle1 - 2 * angle2)));
```

代码就是上面的样子，这里就告诉了我们需要额外定义的变量，基本上就是角加速度角速度，质量等。

随后我们获取每帧的偏差，作为 $dt$ 积分到我们的角度上去。

使用 `GetFrameTime()` 函数获取每一 `tick` 的时间，我们把他累加到函数上去，最后会得到如下的结果。

## 4.2 初始条件

刚刚我们在测试，所以我们把变量全部确定写死在了全剧中，现在我们来给一个随机数，便于每次启动都看到不同的结果。

简单的修改全局变量的位置，以及修改 tick 函数的传入参数，我们就可以实现我们的目的。

随机变量来自于 Raylib 库中的 `GetRandomValue()` 这个函数，同样的，我们可以使用 `SetRandomSeed()` 这个函数设定随机种子。语法同样可以通过 `raylib.h` 查询。

## 4.3 操作优化

我们希望一定程度上的优化操作，加入一些功能，例如使用 `Space` 暂停运算，使用 `R` 来重新开始一次运动。

在 Raylib 中也有相应的控制，我们可以使用 `IsKeyPressed()` 函数来判断对应的按键按下，按键也被封装在了结构体中。

简单的说，结果就是如下的样子。

```cpp
if (IsKeyPressed(KEY_SPACE))
{
	s = (s == Status::RUNNING) ? Status::STOP : Status::RUNNING;
}
if (IsKeyPressed(KEY_R))
{
	angle1 = GetRandomValue(0, 360 * DEG2RAD);
	angle2 = GetRandomValue(0, 360 * DEG2RAD);
}
if (s == Status::RUNNING)
{
	float dt = GetFrameTime();
	tick(dt, angle1, angle1_d, angle1_dd, angle2, angle2_d, angle2_dd);
}
```
