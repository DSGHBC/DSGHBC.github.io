# 关于这个网站

## 实现原理

这个网站使用了 `GitHub Pages + jekyll + Liquid` 的技术来搭建，实际上这个网站的构建是通过 `Markdown` 来实现的。

这将使我把时间和精力花费在网站本身而不是构建网站上。

> Q: 后续是否还会使用这个技术呢?
> 
> A: 不一定，因为我也喜欢好看的 Web 好不好! 所以我可能花费一定的时间和精力去写一个好看的网页! 这个网页先这样吧!

> 配置网站至 Ubuntu 服务器，使用`Nginx`技术。
